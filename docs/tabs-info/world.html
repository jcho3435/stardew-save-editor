World Tab

Change information associated with the world and not associated directly with individual players. Currently supports changing of game day, season, year, weather, and daily luck.

Input value type Value constraints Other notes
Day Integer (non-decimal numeric) Must be a value between 1 and 28 inclusive. -
Season Dropdown selection - -
Year Integer (non-decimal numeric) Must be a value greater than or equal to 1. No known upper contraint. -
Weather Dropdown selection -
  • Windy weather is purely aesthetic. Setting windy weather will not generate debris on your farm.
  • Typically, you cannot have snow if it is not winter, and you cannot have rain during the winter. Using the editor, you should be able to set snow or rain outside of their expected seasons without issue.
  • Setting weather through the editor allows you to bypass days with forced weather. Find info on forced weather on the Stardew Valley Wiki.
  • Currently, stormy weather is not supported. For the time being, setting stormy weather does the same thing as setting rainy weather.
Daily Luck Floating point (decimal numeric) A decimal number between -0.1 and 0.125 inclusive.
  • Daily luck can only naturally be a value between -0.1 and 0.125. For more information on daily luck, check the Stardew Valley wiki.
  • Setting daily luck to a value less than -0.1 or greater than 0.125 is possible, but it comes at the risk of crashing, so it is not recommended. Specific values and scenarios that have been tested:
    • -1.0 luck: No crash when fishing, grabbing artifact spots, or entering Skull Cavern.
    • 1.0 luck: No crash when foraging or grabbing crops. Crash upon entering Skull Cavern.
    • 0.74 luck: Crash upon entering Skull Cavern.
    • 0.2 luck: No crash when fishing, grabbing artifact spots, or entering Skull Cavern
