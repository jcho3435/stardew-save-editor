Friendship Tab

Change friendship data with NPCs for all players who have joined your world. Currently supports changing of friendship points. Use the dropdown menu to swap between farmers. Changed friendship points will be remembered when switching between players.

Input value type Value constraints Other notes
Friendship points Integer (non-decimal numeric) Avoid setting friendship points to a value beyond what is normally possible. Doing some comes at your own risk. You can find information on the normal points values on the Stardew Valley wiki. -
