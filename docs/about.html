About the editor:

This editor is for windows because I don't want to have to think about other operating systems. The code is fully open source and can be found here.

Version info:
Stardew Save Editor Version: v0.2-prerelease
Python version: 3.12.4
Platform: Windows
Port: PySimpleGUI
tkinter version: 8.6.13
PySimpleGUI version: 5.0.6.12

There is no input validation on any of the modifiable fields, so in order to avoid Stardew Valley crashing due to bad save data, avoid inputting bad values. For example, for fields which should take a number, do not input any alphabetical characters. Details on constaints on input values can be found in each tab on the docs. Also, if a field should have a value in it, do not leave it blank. For example, don't leave your farmer name blank.

All error logs and event logs are located in `logs/`
Logs can be cleaned by running `clean_logs.bat`, located in scripts/. This will delete all logs older than 2 weeks.
