Version History

Version 0.2 prerelease - v0.2-prerelease

This is the second prerelease version of Stardew Save Editor! This version adds the ability to modify world data, such as season/year.

Added features:

  • World Tab:
    • Change game date (day, season, and year)
    • Change weather

Other changes:

  • Documentation
    • Added version history page to web docs
    • Added world tab info page to web docs
    • Changed friendship tab info web docs layout
    • Fixed LICENSE link in README.md
    • Added the editor's version to the README.md file and added version info to the bottom of the about tab.
    • Added PySimpleGUI account creation info to web docs.

Version 0.1 prerelease - v0.1-prerelease

This is the first official prerelease version of the Stardew Save Editor!

Features:

  • Backup creation
    • Save files are backed up every time they are loaded. You can find your save files in the backups directory.
    • Backups are never automatically deleted.
  • Backup manager
    • Allows for deleting all backups, deleting all backups of a specific farm, or deleting selected backups.
  • Change profile information for all players who have joined your world, including:
    • Farmer name
    • Farmer skill levels
    • Farmer skill experience
  • Friendship
    • Change friendship points with NPCs for all players who have joined your world.
